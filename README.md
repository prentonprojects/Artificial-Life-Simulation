# Artificial-Life-Simulation
A small project I decided to make in order to simulate an artificial life of its own:

Here, organisms evolve through food competition, reproduction, mutation, and finally survival.

## Current Features
- Organisms are fully autonomous
- Survival is based on food
- Genetic traits
- Mutations can be shared across generations
- Reproduction and death are present
- Live statistics
- Organism Inspection Panel

## Can easily be ran locally
```
pip install -r requirements.txt
python app.py
```

Upgrades could come sooner or later.
